Introduction

This documents the project HybGe-Flo3D. HybGe-Flow3D is a library designed for the solution of fluid flow problems in complex, uncertain and evolving geometries in 2D and 3D.

Dependencies

Building and using HybGe-Flow3D has the following dependencies:

  • Boost C++ Libraries with the following compiled libraries:
    1. Filesystem
    2. System
  • Paralution
  • BLAS/LAPACK

Build

Building HybGe-Flow3D is straighforward assuming the user has installed the dependencies and included them in their path. The following describes the process in Linux.

In the following we assume HybGe-Flow3D source is located at ${HYBGE_SOURCE} and we want to install HybGe-FLow3D in ${HYBGE_ROOT}.

$ cd ${HYBGE_SOURCE}
$ mkdir build && cd build
$ cmake .. -DCMAKE_INSTALL_PREFIX=${HYBGE_ROOT}
$ make install

Done! This installs the folders lib and include in the path ${HYBGE_INSTALL}. Be sure the install directories to the users path before building an example. A bash script to set paths based on ${HYBGE_ROOT} is included in the utilities folder.

Usage

HybGe-Flow3D contains several usage examples in the examples directory which illustrate how to go from problem geometry to problem solution for various multiscale flow investigations.
